Quick Fixes

  • USB not listed: Tap Esc → F10, enable USB boot, use UEFI; disable Secure Boot temporarily if needed.
  • No Wi-Fi / Touch: Install chipset/IO first, then Wi-Fi/Bluetooth, then touchscreen/touchpad; finish with HP Support Assistant.
  • Activation: If the device had Windows before, it usually activates online automatically.
