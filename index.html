Install Windows 10 on HP ProBook x360 — Minimal Guide

Clean, dark, readable. All the steps + official links. Works with any ProBook x360 model (G1–G8, 11/4xx series).

Heads-up
Windows 10 reached end-of-support on October 14, 2025. You can still clean-install it and then add drivers, but consider Windows 11 if your device is compatible. See Microsoft’s install media and ISO pages below.

Steps (Quick Path)

1) Download Windows 10

Use Microsoft’s official ISO page or the Media Creation Tool.

  • Windows 10 download (Media Creation Tool)
  • Direct ISO (if MCT not supported)
  • Create installation media — official guide

2) Create a bootable USB (Rufus)

Plug an 8–16 GB USB. Open Rufus → select ISO → Partition: GPT for UEFI (most HP x360), or MBR for legacy BIOS → Start.

  • Rufus (official) · GitHub

3) Add your drivers to the same USB

After Rufus finishes, open the USB in Explorer and create /Drivers and /Apps. Copy Wi-Fi/Chipset/Graphics/Touchpad/Touchscreen drivers there. (Do not modify /sources or /boot.)

4) Boot the HP ProBook x360 from USB

Power on, repeatedly press Esc for Startup Menu → F9 Boot Options → choose your USB. If the USB doesn’t appear, enter F10 BIOS and ensure USB boot is enabled; disable Secure Boot if needed.

5) Install Windows 10 (clean)

During setup, delete partitions on the target drive (careful: erases data), select the unallocated space, and proceed. After first desktop boot, open your USB’s /Drivers and install drivers in this order:

  • Chipset / Intel Serial IO / MEI
  • Intel/AMD graphics
  • Wi-Fi / Bluetooth
  • Audio (Realtek)
  • Touchpad/Touchscreen (ELAN/Goodix/Synaptics), Hotkeys
  • HP Support Assistant (then run “Check for updates”)

6) Finish & verify

  • Win + X → Device Manager → no devices with yellow marks.
  • Windows Update → get remaining updates.
  • In HP Support Assistant, install any firmware/BIOS/touch updates.

Reference
Don’t know your exact ProBook x360 model?
On Windows, run msinfo32 (System Model) or check the label under the laptop hinge. Then open HP Drivers → enter the model (e.g., “ProBook x360 435 G8” or “x360 11 G3 EE”) and pick Windows 10 64-bit.
Tip: Put ISO + drivers on the same USB
After Rufus creates the bootable USB, you can safely add folders like /Drivers and /Apps. The installer ignores extra files at the root; just don’t change /sources or /boot.
Windows 10 EOL & alternatives FYI
Windows 10 is out of support (Oct 14, 2025). If your ProBook x360 meets requirements, consider installing Windows 11 using the same process with a Windows 11 ISO.

Troubleshooting (HP-specific)

  • USB not listed in F9: In BIOS (F10), enable USB boot, set Boot Mode to UEFI, and temporarily disable Secure Boot.
  • Touchscreen/pen not working: Install chipset + Intel Serial IO first, then ELAN/Goodix touchscreen and HP Hotkey/Quick Launch drivers from your model’s page. Reboot after each group.
  • Wi-Fi missing: Install Intel/Realtek Wi-Fi driver from HP page; some models need Bluetooth separate.
  • Audio issues: Install Realtek audio, then run HP Support Assistant for firmware/BIOS updates.
  • Activation: If the ProBook previously had Windows 10/11, it usually auto-activates online. Sign in and run Windows Update.
